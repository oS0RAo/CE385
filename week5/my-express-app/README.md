Workshop 5

การดึงข้อมูลนักเรียนจะทำการดึงข้อมูลรายชื่อนักเรียนทั้งหมดเมื่อ Client เรียก URL /api/students Server 
จะส่งข้อมูล Array students ทั้งหมดกลับไป

ดึงข้อมูลนักเรียนตาม ID จะทำการค้นหาข้อมูลใน Array ที่มี id ตรงกันผ่าน .find() เมื่อ Client เรียก
URL /api/students/(เลขid)

เพิ่มข้อมูลนักเรียนเมื่อ Client ทำการ POST ใน URL /api/students โดบมีการตรวจสอบข้อมูลโดยจะต้องต้องส่ง
name และ age มาให้ครบ เช่น
{
    "name": "New Student",
    "age": 20
}
โดย .push() จะเพิ่มข้อมูลใหม่เข้าไปใน Array
และ res.json() จะส่งข้อมูลที่เพิ่งสร้างกลับไปให้ Client ดู

แก้ไขข้อมูลนักเรียนเมื่อ Client ทำการ PUT ใน URL /api/students/(เลขid) โดย Client จะแก้ไขชื่อและ
อายุของนักเรียนตามเลขID ที่ระบุได้โดยจะเข้าถึงข้อมูลด้วย students[] แล้วเอาค่าใหม่จาก req.body
ไปทับค่าเดิม

ลบข้อมูลนักเรียนเมื่อ Client ทำการ DELETE ใน URL /api/students/(เลขid) ก็จะทำการลบข้อมูลนักเรียนออกจากระบบ
โดยคำสั่ง .splice(index, 1) จะทำการลบข้อมูลใน Array และแสดงข้อความว่าลบสำเร็จแล้ว
